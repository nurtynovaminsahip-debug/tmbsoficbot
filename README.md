# Transfer Market Brawl Stars Bot

Telegram-бот для приёма и публикации объявлений в игровом сообществе Brawl Stars.

## Функциональность

- 8 типов объявлений: Трансфер, Свободный агент, Завершение карьеры, Поиск скрима, Возвращение карьеры, Приостановление карьеры, Свой текст, Реклама
- Система клубов: создание, управление участниками, роли
- Модерация: администраторы принимают или отклоняют заявки
- Автопубликация одобренных объявлений в Telegram-канал
- Техподдержка с ответами от администраторов
- Панель администратора (`/console`): баны, рассылка, лог
- Панель владельца: статистика, управление пользователями и клубами

## Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/ВАШ_НИК/transfer-market-bot.git
cd transfer-market-bot
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить переменные окружения

Создайте файл `.env` в папке с ботом:

```env
BOT_TOKEN=токен_от_BotFather
ADMIN_GROUP_ID=-1003991176144
OWNER_ID=8225798664
CHANNEL_USERNAME=@TrasnferMarktBrawlStars
```

### 4. Запустить

```bash
python bot.py
```

## Переменные окружения

| Переменная | Описание | Обязательно |
|---|---|---|
| `BOT_TOKEN` | Токен бота от @BotFather | ✅ |
| `ADMIN_GROUP_ID` | ID группы администраторов | ✅ |
| `OWNER_ID` | Telegram ID владельца | ✅ |
| `CHANNEL_USERNAME` | Username канала для публикаций | ✅ |

## Требования

- Python 3.11+
- python-telegram-bot 21.6

## Важно

- Бот должен быть администратором в канале публикаций
- Бот должен быть добавлен в группу администраторов
- Файлы `.env` и `bot.db` не должны попадать в репозиторий
